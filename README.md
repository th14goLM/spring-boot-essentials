## 🚀 Conteúdo

O repositório está organizado contendo:

* 📌 **Fundamentos do Spring Boot**
  * Configuração do projeto com Spring Initializr
  * Estrutura de pacotes e camadas (Controller, Service, Repository, Exception, Utils, Handler, Database)
  * Injeção de dependências

* 📌 **Persistência de dados**
  * Spring Data JPA
  * Mapeamento de entidades
  * Repositórios e consultas

* 📌 **Segurança**
  * Spring Security
  * Autenticação e autorização

* 📌 **Integração e documentação**
  * OpenFeign para chamadas HTTP entre serviços
  * Documentação de API com SpringDoc / Swagger UI

* 📌 **Projetos práticos**
  * APIs REST completas
  * Integração entre múltiplas camadas
  * Boas práticas de desenvolvimento
